## Getting Started

This project require NodeJS 6.14.13.

### Install
Clone this project

```bash
> git clone https://github.com/Elquemao/whatsapp-bot.git
> cd whatsapp-bot
```

Install the dependencies:

```bash
> npm install
```

### Usage
1. run the Whatsapp bot

```bash
> npm start
```

after running it you need to scan the QR with your mobile.
